## 1. Executive Summary

### 핵심 발견

| 지표 | 수치 |
|------|------|
| **Vertical SaaS 성장률** | CAGR 23.9% (Horizontal 대비 2-3배) |
| **1인 Vertical SaaS 평균 MRR** | $5K - $50K |
| **평균 NRR** | 110-120%+ |
| **이탈률** | Horizontal 대비 50% 낮음 |
| **시장 침투율** | 특화 산업 50%+ |

### 핵심 인사이트

```
"Horizontal SaaS는 망치를 판다.
Vertical 1인 SaaS는 특정 지붕용 손톱을 
정확히 박는 도구를 만든다."
```

**Vertical 1인 SaaS 5대 강점**:
1. **깊은 산업 이해** - 워크플로우, 규제, 언어 내재화
2. **낮은 경쟁** - 대기업의 관심 밖 틈새 시장
3. **높은 전환 비용** - 이탈률 극도로 낮음
4. **프리미엄 가격** - 산업 특화 가치 인정
5. **입소문 효과** - 밀집 산업 커뮤니티 내 확산

## 2. Vertical SaaS 개요

### 2.1 정의와 특징

| 구분 | Horizontal SaaS | Vertical SaaS |
|------|-----------------|---------------|
| **타겟** | 모든 산업 | 특정 산업 |
| **예시** | Salesforce, Slack | Toast, Veeva, Procore |
| **커스터마이징** | 높은 필요 | 즉시 사용 가능 |
| **구현 기간** | 3-6개월 | 1-2주 |
| **고객 성공** | 일반적 | 산업 전문가 |

### 2.2 1인 Vertical SaaS 특징

```
┌─────────────────────────────────────────────────────────────┐
│              1인 Vertical SaaS 특징                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ✅ 강점                                                    │
│  • 깊은 산업 지식 = 높은 진입장벽                          │
│  • 소규모 틈새 = 대기업 경쟁 회피                          │
│  • 높은 고객 충성도 = 낮은 이탈률                          │
│  • 입소문 마케팅 = 낮은 CAC                                │
│  • 프리미엄 가격 = 높은 마진                               │
│                                                             │
│  ⚠️ 과제                                                    │
│  • 시장 크기 제한                                          │
│  • 산업 경기 민감도                                        │
│  • 규제 변화 리스크                                        │
│  • 확장성 제한                                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.1 Mangomint - 살롱/스파 운영체제

#### 프로필
| 항목 | 내용 |
|------|------|
| **창업자** | Daniel Lang |
| **국적** | 오스트리아/미국 |
| **시작** | 2016년 (크로스컨트리 여행 중) |
| **현재 ARR** | $10M+ (8자리 수) |
| **NRR** | 110% |
| **월간 예약** | 200,000+ |
| **위치 수** | 1,000+ (미국, 캐나다) |

#### 시작 스토리
```
2016년: 미국 횡단 여행
    ↓
"어디를 가든 살롱과 스파가 있었지만,
 예약하려면 전화해야 했다"
    ↓
불편함 발견 → Mangomint 개발 결심
    ↓
"살롱과 스파를 위한 OS"
```

#### 제품 진화
| 단계 | 기능 | 전략 |
|------|------|------|
| **1단계** | 온라인 예약 | 진입점 (웨지) |
| **2단계** | POS, 결제 | 핵심 업무 |
| **3단계** | 급여, 인사 | 운영 전체 |
| **4단계** | CRM, 마케팅 | 고객 유지 |
| **5단계** | 자동화, AI | 효율화 |

#### 성공 요인
1. **COVID-19 기회**: 노동력 부족 → 자동화 수요 급증
2. **직관적 UX**: "AI라는 단어 없음" - 단순함 추구
3. **수직 통합**: 예약 → 결제 → 급여 → CRM 전체 커버
4. **입소문 성장**: 단 한 번의 콜드콜 없이 성장

> "우리 경쟁사들은 온라인 예약만 생각하지만, Mangomint에서는 AI라는 단어를 찾을 수 없습니다. 우리는 인간을 보조하는 것에 집중합니다."
> — Daniel Lang

## 8. 성공 공식 분석

### 8.1 Vertical 1인 SaaS 성공 공식

```
┌─────────────────────────────────────────────────────────────┐
│           Vertical 1인 SaaS 성공 공식                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. INDUSTRY SELECTION (산업 선택)                          │
│     └── 디지털화가 늦은 전통 산업                          │
│     └── 창업자의 개인적 경험/네트워크 있는 곳              │
│     └── 밀집된 커뮤니티 (입소문 가능)                      │
│     └── 예: 건설, 뷰티, 수의학, 장례업                     │
│                                                             │
│  2. WORKFLOW DEEP DIVE (워크플로우 심층 분석)              │
│     └── 하루 일과 관찰                                     │
│     └── 종이/Excel 사용 지점 파악                          │
│     └── "이게 제일 귀찮아요" 찾기                          │
│     └── 규제/컴플라이언스 요건 확인                        │
│                                                             │
│  3. ONE-WEDGE START (한 워크플로우 시작)                    │
│     └── "올인원"이 아닌 "이것만 완벽하게"                  │
│     └── 예: 예약 관리, 견적 작성, 재고 추적                │
│                                                             │
│  4. WHITE-GLOVE ONBOARDING (고급 온보딩)                    │
│     └── 첫 50명 직접 대면/화상 온보딩                      │
│     └── 데이터 마이그레이션 지원                           │
│     └── 업계 용어 사용한 교육                            │
│                                                             │
│  5. COMMUNITY-LED GROWTH (커뮤니티 주도 성장)              │
│     └── 업계 협회/단체 참여                                │
│     └── 트레이드쇼 부스                                    │
│     └── 고객 성공 사례 공유                                │
│     └── 입소문 유도 (추천 프로그램)                        │
│                                                             │
│  6. LAND & EXPAND (확장)                                    │
│     └── 첫 워크플로우 성공 후 인접 기능 추가               │
│     └── NRR 110%+ 달성                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 9. 시작 가이드

### 9.1 산업 선택 체크리스트

```
✅ 나의 경험
  □ 이 산업에서 일해본 적 있음
  □ 가족/친구가 이 업계 종사
  □ 이 업계 고객과 일해봄

✅ 시장 조건
  □ 디지털화가 늦은 산업
  □ Excel/종이 여전히 사용
  □ 1,000개 이상의 잠재 고객
  □ 연 $50K+ 수익 가능

✅ 접근성
  □ 업계 협회/모임 존재
  □ 트레이드쇼 참여 가능
```
