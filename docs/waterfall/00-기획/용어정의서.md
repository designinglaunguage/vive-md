# 용어정의서

> 프로젝트 용어의 단일 정본이다. 기획 단계에서 먼저 만들고, 이후 SRS·설계·테스트·운영 문서는 이 문서를 링크로 참조한다.

## 1. 작성 규칙

- 새 용어는 다른 문서에 정의하기 전에 이 문서에 먼저 등록한다.
- 기존 문서의 용어 섹션은 삭제하지 않고, 정본 링크를 함께 둔다.
- 같은 의미의 용어가 둘 이상이면 대표 용어 하나를 정하고 나머지는 별칭에 적는다.
- 정의는 구현자가 같은 동작을 만들 수 있을 만큼 관찰 가능하게 쓴다.
