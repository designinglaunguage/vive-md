## 📋 KBW 주간브리프 형식

### 파일명 형식
```
[주간브리프]KBW_YYMMDD-NN.md
```
- `YYMMDD`: 날짜 (예: 260209 = 2026년 2월 9일)
- `NN`: 버전/차수 (01부터 순차적)

- 배터리 기술/생산
- 신기술 개발

## 총 건수: 12-18건

3. 수치/데이터는 `• `로 bullet point
4. 출처는 `☞ 출처: [언론사] [YYYY/MM/DD]`
5. 원문 링크는 Markdown 형식으로 제공

## 검색 대상

## 📂 섹션별 설명

### [정책/시장 동향]

**포함 내용:**
- 각국 정부의 EV 관련 정책
- 보조금, 세제 혜택
- 환경/안전 규제
- 무역 정책 (관세, FTA)
- 시장 동향 (판매 실적, 점유율)

**추천 사이트:**
- 중국: 有驾, 36氪
- 미국: Bloomberg, Reuters
- 한국: 머니투데이, 전자신문
- 일본: 日経クロステック

**예시:**
```markdown
□ 캐나다, 2035년까지 전기차 100% 계획 철회
캐나다 마크 카니 총리는 5일 자동차 산업 개편 방안을 발표
2035년 신차 판매 100% 전기차 의무화 계획을 폐기
2035년 75%, 2040년 90%를 새 목표치로 제시
BEV 최대 5천캐나다달러, PHEV 최대 2500캐나다달러 보조금 부활
☞ 출처: [electrive](https://...) 2026/2/6
```

---

### [전방산업 동향]

**포함 내용:**
- 완성차 업체(OEM) 동향
- 신차 출시/공개
- 생산/판매 계획
- 투자/M&A
- 공장 설립/확장

**추천 사이트:**
- 중국: 盖世汽车, 汽车之家
- 미국: Automotive News, The Verge
- 한국: 모터그래프, 오토타임즈
- 일본: Response.jp, Car Watch

**예시:**
```markdown
□ Toyota, 2028년 하이브리드 생산량 670만대 목표
2028년 목표치는 2026년 목표치인 500만대보다 30% 높은 수치
2025년 미국 시장 총 판매량 251만 대(YoY +8%)
향후 5년간 미국에 100억달러 투자
하이브리드 차량 비중은 2026년 약 50%에서 2028년 약 60%로 늘어날 전망
☞ 출처: [Automotive World](https://...) 2026/2/4
```

---

### [업체/기술 동향]

**포함 내용:**
- 부품/소재 업체 동향
- 배터리 기술/생산
- 신기술 개발 (SDV, 자율주행 등)
- 공급망 동향
- 투자/합작

**추천 사이트:**
- 중국: 盖世汽车
- 미국: Electrek, InsideEVs
- 한국: 전자신문, 디일렉, ZDNET
- 일본: 日経クロステック, MarkLines

**예시:**
```markdown
□ Panasonic, 캔자스 배터리 공장 생산량 50% 근접
Panasonic의 미국 캔자스 배터리 공장이 몇 달 안에 생산 능력의 50%까지 가동률을 끌어올릴 예정
기존에 가동 중인 2개 라인에 더해, 2개의 새로운 생산 라인 가동을 시작
Second Wing 건설이 시작되었으며, 해당 구역의 4개 라인 중 2개는 2027년에 가동될 예정
☞ 출처: [Kansas Reflector](https://...) 2026/2/3
```
