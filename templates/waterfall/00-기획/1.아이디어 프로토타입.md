```mermaid
flowchart TD
    A[사용자 유입] --> B[가치 제안 확인]
    B --> C{관심 있음?}
    C -->|예| D[신청/문의/결제 의사 표시]
    C -->|아니오| E[이탈]
    D --> F[후속 인터뷰 또는 수동 제공]
    F --> G[학습 내용 기록]
```
