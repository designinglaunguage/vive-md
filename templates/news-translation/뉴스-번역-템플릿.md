# KBW 주간브리프 뉴스 번역 템플릿

> KBW 주간브리프 형식에 맞춘 글로벌 자동차/모빌리티 뉴스 번역 템플릿입니다.
> 출처 링크를 반드시 포함하며, [정책/시장 동향], [전방산업 동향], [업체/기술 동향] 3개 섹션으로 구성됩니다.

---

## KBW 주간브리프 형식

### 파일명 형식
```
[주간브리프]KBW_YYMMDD-NN.docx (또는 .md)
```
- YYMMDD: 날짜 (예: 260209 = 2026년 2월 9일)
- NN: 버전/차수 (예: 06)

### 작성 예시

```markdown
□ 캐나다, 2035년까지 전기차 100% 계획 철회

BEV 최대 5천캐나다달러, PHEV 최대 2500캐나다달러
보조금은 5년간 유지되며, 2030년 BEV 2천캐나다달러, PHEV 1천캐나다달러로 단계적 축소

충전시설 확충에 15억캐나다달러 지원
자동차 산업이 적응, 성장하고 신규시장 다변화를 하도록 돕기 위해 30억캐나다달러 지원

```

## 작성 규칙

### 1. 제목 작성
- `□ `(사각형 기호 + 공백)로 시작
- 한국어로 간결하게 작성
- 핵심 내용이 드러나도록 작성

### 2. 본문 작성
- 3~5문장으로 핵심 내용 요약
- 원문의 중요 정보를 빠짐없이 포함
- 수치/데이터는 bullet point(•)로 정리
- YoY, MoM 등 비교 수치 명시

### 3. 출처 표기
- 반드시 `☞ 출처: ` 형식 사용
- 언론사명 + 날짜 (YYYY/MM/DD 형식)
- 원문 링크는 별도로 관리 (또는 Markdown 링크 형식)

### 4. 섹션 분류 기준

| 섹션 | 포함 내용 | 예시 |
|------|----------|------|
| **[정책/시장 동향]** | 정부 정책, 규제, 시장 동향, 보조금 | EV 의무화, 관세, 판매 실적 |
| **[전방산업 동향]** | 완성차 업체 동향, 신차, 투자 | Toyota 생산 계획, Ford 파트너십 |
| **[업체/기술 동향]** | 부품사, 배터리, 소재, 신기술 | Panasonic 공장, CATL 기술 |

## 주의사항

1. **출처 링크 필수**: 모든 뉴스는 반드시 출처 표기
2. **날짜 형식 통일**: YYYY/MM/DD 형식 사용
3. **수치 정확성**: 원문의 수치를 그대로 인용
4. **섹션 분류 명확히**: 정책/시장 vs 전방산업 vs 업체/기술 구분
5. **간결한 문장**: 3~5문장으로 핵심만 요약
